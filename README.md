TP1
¿Por qué es conveniente incluirlo?
Para evitar la lectura no necesaria de archivos _generados_ por los programas que utilizamos a diario.
¿Cuándo se debe hacer?
Cuando nos vamos a centrar en algunos archivos especificos o solo nos interesa visualizar de manera mas **oredenada** los archivos.
¿Cómo configuraría el archivo .gitignore?
Se abre con un editor de texto y se agregan "reglas" que especifican que archivos se van a tener en cuenta y cuales vas a a ser ignorados.
